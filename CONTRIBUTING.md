# Contributing
Obrigado por considerar contribuir com este projeto!
Pull requests serão aceitos a partir de 24/02/2026.

## Como contribuir
* Faça um fork do repositório
* Crie uma branch para sua feature:

  ```
  git checkout -b minha-feature
  ```
* Faça suas alterações
* Commit:

  ```
  git commit -m "feat: minha nova feature"
  ```
* Envie para o repositório:

  ```
  git push origin minha-feature
  ```
* Abra um Pull Request


## Boas práticas
* Mantenha o código organizado e legível
* Siga o padrão de programação orientada a objetos
* Evite código duplicado
* Comente apenas o necessário (sem excesso)


## Testes
* Sempre que possível, adicione testes na pasta `testes/`
* Certifique-se de que tudo compila com:

  ```
  make
  ```

## Relatar problemas
* Use a aba *Issues*
* Descreva claramente o problema encontrado
